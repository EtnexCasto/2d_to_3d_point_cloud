# 3D фотограмметрия

Преобразование фотографий в 3D облако точек с помощью фотограмметрии и компьютерного зрения.

## Возможности

- Загрузка нескольких фотографий
- Автоматическая 3D реконструкция (SIFT + SfM + MVS)
- Визуализация облака точек в браузере
- Скачивание результата в формате PLY

## Установка

### 1. Установите COLMAP

- Windows: Скачайте с https://colmap.github.io/ и добавьте в PATH
- Linux: `sudo apt-get install colmap`
- macOS: `brew install colmap`
